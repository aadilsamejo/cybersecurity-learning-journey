# Networking Task 01 - Basic LAN Communication

## Objective

Create a simple LAN using Cisco Packet Tracer and test communication between two devices.

## Topology

PC1 ----- Switch ----- PC2

## IP Configuration

PC1: 192.168.1.10

PC2: 192.168.1.20

Subnet Mask: 255.255.255.0

## Testing

Used ping to test connectivity.

Result:

- 4 packets sent
- 4 packets received
- 0 packet loss

## Skills Learned

- IP Addressing
- LAN Communication
- Switch Operation
- Ping Command
